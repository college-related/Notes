Bisection Method

  • Formula

  1. Given function f(x),
  2. we run it by 0,1,2,3,… => f(0),f(1),f(2),f(3),…
  3. The points when the result changes the sign are the roots (a,b)
  4. We use the formula
    x=(a+b)2 x = \frac{(a+b)}{2} x=2(a+b)​
  5. We fill the table
Step a b x=(a+b)2x=\frac{(a+b)}{2}x=2(a+b)​ f(x)
  1. In each step we check
    f(x)=+ve  or  f(x)=−veif  (f(x)=+ve) then put b=x in next stepif  (f(x)=−ve) then put a=x in next step f(x) = +ve \space\space or \space\space f(x) = -ve \\ if \space\space (f(x) = +ve) \space then \space {put \space b = x \space in \space next \space step} \\ if \space\space (f(x) = -ve) \space then \space {put \space a = x \space in \space next \space step} f(x)=+ve  or  f(x)=−veif  (f(x)=+ve) then put b=x in next stepif  (f(x)=−ve) then put a=x in next step
  2. Repeat the step 5 and 6 till we get repeating value of x, which is the required root of the function

  2. We run it by 0,1,2,… => f(0),f(1),f(2),…

  4. We use the formula
    x=a∗f(b)−b∗f(a)f(b)−f(a) x = \frac{a*f(b)-b*f(a)}{f(b)-f(a)} x=f(b)−f(a)a∗f(b)−b∗f(a)​

Newtons Raphson Method

  • Formula

  1. Given f(x)
  2. Find f’(x)
  3. We run it by 0,1,2,… => f(0), f(1), f(2),…
  4. Root lies between (a,b) i.e the points where the sign changes
  5. Put x0 = a
  6. We use the formula
    xn+1=xn−f(xn)f′(xn) x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)} xn+1​=xn​−f′(xn​)f(xn​)​
  7. We fill the table
n xn f(xn) f’(xn) xn+1=xn−f(xn)f′(xn)x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}xn+1​=xn​−f′(xn​)f(xn​)​
  1. Fill the table till the value of xn+1 repeats

  • Limitation

    1. If f’(xn) is 0, then divisible by 0 occurs which cannot give the solution
    2. If the initail guess is not a good guess, i.e too far from exact root then the process may converge to a garbage root.

Curve fitting (Least square method)

  • Formula

  1. Given function f(x)
  2. Given table
x x1x_1x1​ x2x_2x2​ x3x_3x3​ … xnx_nxn​
y y1y_1y1​ y2y_2y2​ y3y_3y3​ … yny_nyn​
  1. Convert the given function f(x) into form of y = a + bx [Mostly by taking logelog_eloge​ on both side of f(x)]
  2. We then find ∑x,∑y,∑xy,∑x2\sum x, \sum y, \sum xy, \sum x^2∑x,∑y,∑xy,∑x2 using the given table
  3. Construct the normal equation of constant a and b as
    ∑y=na+b∑x−−−−i−−−−∑xy=a∑x+b∑x2−−−ii \sum y = na + b \sum x ---- i \\ ----\\ \sum xy = a \sum x + b \sum x^2 --- ii \\ ∑y=na+b∑x−−−−i−−−−∑xy=a∑x+b∑x2−−−ii
  4. Solve equation i and ii and find value of a and b
  5. Finally substitue the value of a and b in the equation y = a + bx
